# hasattr(node, "text") 
> node라는 변수(객체) 안에 'text'라는 이름의 데이터나 기능이 들어있니?" 라고 물어보는 것과 같습니다.
> 

# getattr(node, "text", "No Text") 
> text 속성이 없으면 기본값으로 "No Text"를 반환

# re.match(r"note\s*\d", node_text
> node_text 가 반드시 "note 1", "Note2" 처럼 'note' 로 시작 해야만 인식을 합니다. 
만약 "Important note 1" 처럼 앞에 다른 글자가 있으면 찾지 못합니다.

# re.search(note_pattern, node_text)
> re.search() 는 문자열 '전체' 를 훑어서 패턴이 어디에 있든 처음 발견되는 곳을 찾아냅니다.
> node_text 의 앞, 중간, 뒤 어디든 note_pattern 이 포함되어 있으면 성공입니다.
예를 들어 패턴이 "Risk" 라면, "Financial Risk Report" 라는 문자열에서 중간에 있는 "Risk" 를 찾아냅니다.
